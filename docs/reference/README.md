# Section Reference

Each Phase 2 section ships with a contract:

- **Markup structure** — required HTML hierarchy
- **CSS classes** — root + element + modifier classes
- **Token hooks** — overridable design tokens
- **Variants** — bounded variant list
- **Accessibility** — keyboard, ARIA, contrast guarantees
- **Composition** — what it composes (button, container, card)
- **Container queries** — responsive thresholds
