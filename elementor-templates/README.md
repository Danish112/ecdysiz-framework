# Ecdysiz Elementor Templates

Production-ready Elementor section templates. Drag-drop or import per project.

## Workflow

1. Build section in Elementor with `ecdysiz-` classes
2. CSS styles live in `theme/ecdysiz-core/assets/css/sections.css`
3. Export Elementor template → save JSON here
4. Document in `docs/reference/sections/<name>.md`
5. New client project: import JSON → swap content → done
